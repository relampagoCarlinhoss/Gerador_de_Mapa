# Gerador De Mapas
Este projeto foi criado durante a disciplina de Introdução a Técnicas de Programação(ITP). Consiste basicamente numa aplicação que gera uma mapa quadrado com relevo. 

---
## 🛠️ Tecnologias e Ferramentas

* **Linguagem Principal:** C++ (Padrão C++ 20) 
* **Editor/IDE:** VScode
